Hugo 0.60 では記事からHTMLが取り除かれる

2021-01-24

web hugo

そういえば対処していなかった。

Hugo 0.60 ではMarkdown処理系が代わったため、HTMLを直に記載していると該当部分が

<!-- raw HTML omitted -->

となってしまいます。

config.toml の [markup] 設定を下記のように書き換えると、元の動作に近い結果に。

[markup]
[markup.goldmark]
[markup.goldmark.renderer]
unsafe = true

相変わらず Hugo は漫然とアップデートしていると刺される。
